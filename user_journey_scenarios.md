# 5. システムの価値・メリット確認

## 薬局側のメリット

```
✅ 短期で確実に薬剤師を確保できる
- 産休・育休・退職などの急な欠員に対応
- 体験期間で相性確認できる

✅ リスクが少ない
- 手数料先払いで薬剤師が保護されている
- 合わなければ30日で終了

✅ 採用コストが明確
- 日給 × 日数 + 手数料40%
- 追加費用なし

✅ 正式雇用への道
- 体験期間で見極め
- そのまま正式雇用も可能
```

## 薬剤師側のメリット

```
✅ 様々な薬局を経験できる
- 短期で色々な職場を知れる
- 自分に合う職場を見つけられる

✅ 報酬が保証されている
- 手数料先払いで保護されている
- 確実に報酬を受け取れる

✅ 正式雇用のチャンス
- 体験期間で薬局を見極め
- 気に入れば正式雇用

✅ 柔軟な働き方
- 15〜90日の期間選択
- 転職活動中のつなぎにも
```

## 運営側のメリット

```
✅ 手数料収入
- 契約成立ごとに40%の手数料

✅ 薬局・薬剤師の双方にメリット
- Win-Winの仕組み

✅ スケーラブル
- システム化で効率運営
- 管理者の負担は証明書確認と支払い確認のみ
```

# 6. 重要なチェックポイント

## システムが機能するための必須条件

```
1. 【管理者】資格証明書の迅速な確認
   → 薬剤師が応募できるようになる
   → 遅延すると機会損失

2. 【管理者】支払いの迅速な確認
   → 個人情報が開示される
   → 契約が実行に移る

3. 【薬局】手数料の期限内支払い
   → 支払いがないと契約キャンセル
   → ペナルティ適用

4. 【システム】自動化された処理
   → 契約書類の自動生成
   → 期限管理・通知
   → ペナルティの自動適用
```
