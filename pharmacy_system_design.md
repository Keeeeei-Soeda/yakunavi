# 薬局管理システム：薬局側機能設計書

## 1. 概要

### 1.1 システムの目的
薬局が薬剤師を短期〜中長期（10〜90日間）で募集・採用するためのマッチングプラットフォーム

### 1.2 主要な特徴
- 日給固定制（2.5万円/日）
- 体験期間での相性確認
- プラットフォーム手数料先払いによる薬剤師保護
- 労働条件通知書の自動発行

## 4. 求人管理機能

### 4.1 求人投稿

#### 4.1.1 入力フォーム

**必須項目**
```
1. 求人タイトル *
   ├─ 入力形式：テキスト（最大100文字）
   └─ 例：「急募！大阪市中央区の調剤薬局で薬剤師募集」

2. 求人詳細
   ├─ 入力形式：テキストエリア（最大2000文字）
   ├─ 内容：仕事内容、職場の雰囲気、設備など
   └─ プレースホルダー：
       「例：
       - 主な業務内容
       - 職場の雰囲気
       - 設備・システム
       - その他アピールポイント」

3. 勤務地 *
   ├─ 入力形式：テキスト
   └─ 例：「大阪市中央区」

4. 日給 *
   ├─ 入力形式：数値入力（20,000円以上）
   ├─ バリデーション：20,000以上
   ├─ デフォルト値：25,000円
   ├─ 補足テキスト：「日給は20,000円以上で設定してください」
   └─ 例：20,000円、22,000円、25,000円、30,000円

5. 希望勤務日数 *
   ├─ 入力形式：数値入力（15〜90）
   ├─ バリデーション：15以上90以下
   ├─ リアルタイム計算表示：
   │  「報酬総額：660,000円（日給22,000円 × 30日）」
   │  ※日給と日数の入力に応じて自動計算
   └─ ヒント表示：
       「💡 長期間（60〜90日）または短期間で高単価の求人は
       応募が集まりやすい傾向があります」

6. 勤務開始可能期間 *
   ├─ 入力形式：日付選択
   ├─ 制限：今日から2週間後以降の日付のみ
   ├─ ラベル：「この日から2週間の間で初回勤務日を相談」
   └─ 補足テキスト：
       「※薬剤師と相談の上、初回勤務日を決定します」

7. 募集期限 *
   ├─ 入力形式：日付選択
   ├─ デフォルト値：今日から7日後
   ├─ 制限：最短3日後、最長14日後
   └─ バリデーション：3〜14日の範囲

8. 応募条件・資格 *
   ├─ 入力形式：テキストエリア（最大500文字）
   └─ 例：「薬剤師免許必須、調剤経験3年以上」
```

**任意項目**
```
9. 希望勤務時間帯
   ├─ 入力形式：テキストエリア
   ├─ プレースホルダー：「例：平日9:00-18:00希望」
   └─ 補足テキスト：
       「※あくまで希望です。実際の勤務時間は
       薬剤師と相談の上決定します」
```

#### 4.1.2 確認画面
```
├─ 入力内容のプレビュー表示
├─ 報酬総額の再確認
├─ プラットフォーム手数料の表示
│  └─ 「手数料：◯◯円（報酬総額の40%）」
└─ 投稿ボタン
```

#### 4.1.3 投稿後
```
├─ 求人公開
├─ 求人一覧に追加
├─ 確認通知の送信
└─ ダッシュボードへリダイレクト
```

### 4.2 求人一覧・管理

```

・前日までに未払いの場合、契約は自動キャンセルされます
・薬剤師への報酬は体験期間終了後に直接お支払いください
```

#### 6.3.2 送信後の処理
```
1. 薬剤師に通知
2. 薬剤師の承認待ちステータスに変更
3. メッセージエリアに「正式オファーを送信しました」と表示

```

## 付録B：ステータス一覧

### 求人ステータス
```
├─ 下書き：保存されているが未公開
├─ 募集中：公開され応募受付中
├─ 募集終了：募集期限到達
├─ 契約成立：薬剤師が決定
└─ キャンセル：募集中止
```

### 応募ステータス
```
├─ 未読済：新規応募、未確認
├─ 検討中：応募を確認済み
├─ オファー送信済：オファーを送信
├─ 辞退：薬局側が辞退
└─ 薬剤師辞退：薬剤師側が辞退
```

### 契約ステータス
```
├─ 承認待ち：正式オファー送信、薬剤師の承認待ち
├─ 手数料支払い待ち：薬剤師承認済み、手数料未払い
├─ 契約成立：手数料支払い完了、連絡先開示
├─ 勤務中：初回出勤日以降
├─ 契約終了：体験期間終了
└─ キャンセル：手数料未払いによる自動キャンセル
```
