# 薬局管理システム：薬剤師側機能設計書

## 1. 概要

### 1.1 システムの目的
薬剤師が短期〜中長期（10〜90日間）の求人を検索・応募し、薬局とマッチングするためのプラットフォーム

### 1.2 主要な特徴
- 日給固定制（2.5万円/日）
- 資格証明書による本人確認
- 体験期間での相性確認
- 個人情報保護（手数料支払い確認後に開示）

---

## 2. 薬剤師アカウント管理

### 2.1 新規アカウント作成

#### 入力項目
```
【薬剤師として登録】

【認証情報】
├─ メールアドレス *
└─ パスワード *（8文字以上、英数字含む）
└─ パスワード確認 *

【基本情報】
├─ 姓 *
├─ 名 *
├─ 電話番号 *
└─ 住所 *（運営側でのみ確認、薬局側には非開示）

【資格情報】
├─ 薬剤師免許番号 *
└─ 経験年数 *（選択式：0年、1年、2年、3年、4年、5年以上）

【利用規約】
└─ 利用規約に同意します * (チェックボックス)
```

#### バリデーション
- メール形式チェック
- パスワード強度チェック（8文字以上、英数字含む）
- 電話番号形式チェック
- 薬剤師免許番号の形式チェック
- 必須項目の入力チェック

#### 登録フロー
```
1. 入力フォーム表示
2. バリデーション実行
3. アカウント作成
4. 確認メール送信
5. メール認証
6. ログイン可能に

⚠️ この段階では応募不可
⚠️ 資格証明書のアップロードが必須
```

## 12. 応募キャンセルについて

### 12.1 キャンセルポリシー

```
【応募後のキャンセル】

❌ 薬剤師側からの直接キャンセル機能なし

⚠️ キャンセルが必要な場合：
「応募をキャンセルされる場合は、運営（support@yakunavi.jp）
までご連絡ください。運営にて確認の上、対応いたします。」

理由：
・薬局側への影響を最小限にするため
・安易なキャンセルを防止するため
・運営が状況を把握し、適切に対応するため

⚠️ 重要：一度応募したら、基本的に取り下げはできません
```

## 付録B：ステータス一覧

### 資格証明書ステータス
```
├─ 未アップロード：証明書がアップロードされていない
├─ 確認中：運営による確認中
├─ 確認完了：本人確認完了、応募可能
└─ 差し戻し：証明書に不備があり、再アップロード必要
```

### 応募ステータス（薬剤師側から見た）
```
├─ 応募済み：応募完了、薬局の返信待ち
├─ メッセージ中：薬局とやり取り中
├─ 日程調整中：初回出勤日を調整中
├─ オファー受信：正式オファー受信、承認待ち
├─ 契約成立：オファー承認、薬局の手数料支払い待ち
├─ 勤務予定：手数料支払い完了、勤務開始待ち
└─ 契約終了：体験期間終了
```

### 契約ステータス（薬剤師側）
```
├─ 薬局の手数料支払い待ち：契約成立、薬局の手数料支払い待ち
├─ 契約成立：手数料支払い完了、連絡先開示済み
├─ 勤務中：初回出勤日以降
├─ 契約終了：体験期間終了
└─ キャンセル：手数料未払いによる自動キャンセル
```
