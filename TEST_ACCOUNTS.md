## 📌 テスト手順

### 1. 薬剤師側での応募テスト

1. **薬剤師アカウント1でログイン**
   - メール: `pharmacist1@test.com`
   - パスワード: `password123`

2. **求人検索ページにアクセス**
   - `/pharmacist/jobs`

3. **求人詳細を確認**
   - 求人1または求人2をクリック
   - 詳細ページで情報を確認

4. **応募する**
   - 「応募する」ボタンをクリック
   - 応募確認ダイアログで以下を入力:
     - 最寄駅: 新宿駅（自動入力済み）
     - 勤務経験のある業態: 調剤薬局、ドラッグストア（チェック）
     - 自己紹介: 任意
   - 「応募する」をクリック

### 2. 薬局側での応募確認テスト

1. **薬局アカウント1でログイン**
   - メール: `pharmacy1@test.com`
   - パスワード: `password123`

2. **応募管理ページにアクセス**
   - `/pharmacy/applications`

3. **応募者を確認**
   - 応募一覧で応募者を確認（匿名表示）
   - 詳細ページでプロフィールを確認

4. **メッセージを送信**
   - 応募者詳細ページからメッセージ画面へ
   - 初回出勤日の候補を提案

### 3. メッセージ・契約フローテスト

1. **薬剤師側でメッセージ確認**
   - 薬剤師アカウント1でメッセージ画面にアクセス
   - 薬局からのメッセージを確認
   - 初回出勤日を選択

2. **薬局側で正式オファー送信**
   - 薬局アカウント1でメッセージ画面にアクセス
   - 契約条件を確認
   - 「正式オファーを送信」をクリック
   - 契約内容を入力・確認
   - オファーを送信

3. **薬剤師側でオファー承認**
   - 薬剤師アカウント1でメッセージ画面にアクセス
   - 「正式オファーを確認」ボタンをクリック
   - 契約内容を確認
   - 「承認」をクリック
